VERSÃO: 1.0

OBJETIVO

Este documento organiza as 152 perguntas dos roteiros de imersão pela perspectiva do stakeholder — ou seja, agrupa todas as perguntas que devem ser feitas a cada perfil de entrevistado, independente do roteiro de origem.

O consultor deve usar este documento para preparar cada entrevista individual, garantindo que todas as perguntas relevantes para aquele stakeholder sejam cobertas em uma única sessão (ou no menor número possível de sessões).

COMO USAR ESTE DOCUMENTO

1. No Momento 1 (Planejamento), o consultor identifica quais stakeholders estão disponíveis.
2. Consulta o cenário de aplicação compatível (Seção 1).
3. Monta o guia de cada entrevista usando as seções de stakeholder correspondentes (Seção 2).
4. As perguntas marcadas como NUCLEO_ESSENCIAL devem ser priorizadas. As COMPLEMENTAR podem ser cortadas se o tempo for insuficiente.

SEÇÃO 3 — REGRAS DE ABSORÇÃO
════════════════════════════════════════

Quando um stakeholder não está disponível, suas perguntas são absorvidas por outro. A tabela abaixo define a regra de absorção:

STAKEHOLDER INDISPONÍVEL → ABSORVIDO POR

Diretor de Operações → CEO
Diretor Financeiro → CEO
Colaborador antigo / RH → Imersão fica sem triangulação de cultura (registrar como lacuna)
Diretor Comercial → CEO absorve ROT_MERCADO e ROT_PORTFOLIO; Responsável por Marketing absorve ROT_PUBLICO
Responsável por Marketing → CEO absorve ROT_MKTVENDAS (qualidade reduzida — registrar como lacuna)
CEO → Imersão é inviável. Reagendar.
